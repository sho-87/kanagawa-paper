# Kanagawa Paper

![Kanagawa](assets/logo.png)

Visual Studio Code port of the [kanagawa.nvim](https://github.com/rebelot/kanagawa.nvim) theme.

## Paper

This version is my own custom remix that I use in nvim and have ported over to VSCode.

It uses the original color palette as reference, but contains some modifications to make the overall feel more muted and pastel-like.

Achieved by combining the syntax colors of the `Dragon` theme with the UI colors of the `Wave` theme.

Link: [Theme Studio](https://themes.vscode.one/theme/DrJellybean/Qr8EIKyy)
